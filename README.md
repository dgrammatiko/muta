# For the package visit the [site](https://muta.dgrammatiko.dev)


### Commands

- Initialize `npm run init`

Then:

- Build `npm run build`
- Link `npm run link`

## Commands
```
Fetch Joomla:       "get-joomla"

Initialise the dev: "init"
Lint css:           "lint-css"
Lint js:            "lint-js"
Lint all:           "lint"
Update browserlist: "update:browserlist"
```
